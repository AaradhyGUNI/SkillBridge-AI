# SkillBridge AI

SkillBridge AI is an AI-powered Resume Analyzer that helps users evaluate ATS compatibility, identify skill gaps, receive personalized learning recommendations, and generate interview questions using Google Gemini AI.

## Technologies Used

### Frontend

* React.js
* Vite
* JavaScript
* HTML5
* CSS3
* Tailwind CSS

### Backend

* Python
* Flask
* Flask-CORS
* REST API

### Artificial Intelligence

* Google Gemini AI

### Development Tools

* Git
* GitHub
* Visual Studio Code

## Key Features

* Resume Upload and Analysis
* ATS Score Evaluation
* Skill Gap Detection
* Personalized Learning Recommendations
* AI-Generated Interview Questions
* PDF Resume Processing
* Responsive User Interface

## Project Objective

The objective of SkillBridge AI is to assist students and job seekers in improving their resumes, identifying missing skills, and preparing for interviews through AI-powered analysis and recommendations.
